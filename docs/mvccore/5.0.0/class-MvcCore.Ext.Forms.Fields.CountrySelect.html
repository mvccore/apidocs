Class CountrySelect

Responsibility: init, pre-dispatch and render <select> HTML element as roll-out menu for single option select or as options list for multiple selection with options as all existing world states or only filtered world states. CountrySelect field has it's own validator to check if submitted value is presented in configured options by default.
